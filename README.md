# **Дипломный проект. Задание 3. Автотесты для UI**

**Проверка основной функциональности**

**Проверить:**

1. переход по клику на «Конструктор»;

2. переход по клику на раздел «Лента заказов»;
3. если кликнуть на ингредиент, появится всплывающее окно с деталями;
4. всплывающее окно закрывается кликом по крестику;
5. при добавлении ингредиента в заказ счётчик этого ингредиента увеличивается.
6. Раздел «Лента заказов»;

**Проверить:**

7. при создании нового заказа счётчик «Выполнено за всё время» увеличивается;
8. при создании нового заказа счётчик «Выполнено за сегодня» увеличивается;
9. после оформления заказа его номер появляется в разделе «В работе».
